Helping Rudolph

Rudolph has got the special mission to deliver presents to the houses in his village.

The elfs gave him instructions with the moves that he has to take such that his route respects the following conditions:

  1. The village is a single row of infinitely many houses, which means Rudolph can only move to the left or to the right.
  2. His start and end point is Santa’s warehouse.
  3. Santa’s warehouse is the first house in the row, which means Rudolph can’t move to the left if he is at the warehouse.
  4. The sequence is composed of “R” and “L” for right and left moves, respectively.

Some valid sequences of moves are "RRRLRLLL", "RRLL" and "RRLRRLLL". "LR", "LLR", and "RLLR" are not valid because they break conditions 2 and 3. Rudolph is now set to start his mission. However, upon looking over the route, he finds that the elfs gave him wrong instructions. Because he doesn’t have time to ask the elfs for new instructions, he decides to form a new route the following way: he takes the longest substring from the old instructions and this substring should be a valid sequence that he can follow. Rudolph is asking you to do the hard part, that is to find the longest substring that represents a valid sequence of moves. You also have to find the total number of such substrings.

Input/Output Description

Input

There is one input parameter: a string \(s\) consisting of 'R' and 'L' characters.

Output

Your function should return a pair of two integers \((n, m)\) where \(n\) is the length of the longest substring that is a valid sequence of moves and \(m\) is the number of such substrings. If there are no valid substrings, return \((0, 1)\).

Constraints

  • \(0 \le |s| \le 50000\)

Examples

Input

  1. “LRRRLLLLRRLRLL”
  2. “LLR”

Output

  1. (6, 2)
  2. (0, 1)
