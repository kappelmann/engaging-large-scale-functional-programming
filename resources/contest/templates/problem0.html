Team Building

The FPV-Übungsleitung is organising an excellent programming contest with only 4 participating teams. The structure of the teams is as follows:

The first team has \(n_1\) participants, the second one has \(n_2=kn_1\) participants, the third team has \(n_3=kn_2\) participants, and the fourth team has \(n_4=kn_3\) participants. The total number of participants to the contest is \(n\), that is \(n=n_1+n_2+n_3+n_4\).

The FPV-Übungsleitung knows the number of registered students and the value of \(k\). Help them to compute the team sizes \(n_1,n_2,n_3,n_4\).

Input/Output Description

Input

The function parameter is a pair of integers \((n, k)\). It is guaranteed that \(n\) and \(k\) are chosen in a way such that the Übungsleitung can successfully assign teams with the sizes as described above.

Output

You should return \((n_1,n_2,n_3,n_4)\).

Constraints

  • \(4 \le n \le 3.3 * 10^{20}\)
  • \(1 \le k \le 200000\)

Example

contestTeams (40, 3) = (1, 3, 9, 27)
